# DHL Nigeria Air Shipping Performance Dashboard

An interactive Power BI dashboard built to analyze domestic DHL air shipments across Nigeria.  
The report tracks shipment volume, revenue performance, delivery status, route efficiency, and delay trends to support operational decision making.
